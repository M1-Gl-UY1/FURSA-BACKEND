## Récapitulatif du flux complet validé 

```
GET  /api/distribution/1              → Calcul dividendes → BD: 3x VALIDE
POST /api/blockchain/investors        → Ajout Alice, Bob, Charlie au contrat
POST /api/blockchain/distribute/blockchain/1 → 
     ├── payInvestor(wallet, Wei)     → Blockchain: txHash généré
     ├── dividende → PAYE + txHash   → BD: dividende mis à jour
     ├── Paiement(CRYPTO, PAYE)      → BD: paiement créé
     └── Transaction(txHash, id_paie)→ BD: transaction enregistrée
```

Si tu veux tout remettre à zéro y compris les données métier :

```sql
-- Suppression complète (dans l'ordre des dépendances)
DELETE FROM transaction;
DELETE FROM paiement;
DELETE FROM dividende;
DELETE FROM possession;
DELETE FROM revenus;

-- Réinitialisation des séquences
ALTER SEQUENCE transaction_id_trans_seq RESTART WITH 1;
ALTER SEQUENCE paiement_id_paie_seq RESTART WITH 1;
ALTER SEQUENCE dividende_id_div_seq RESTART WITH 1;
ALTER SEQUENCE possession_id_pos_seq RESTART WITH 1;
ALTER SEQUENCE revenus_id_rev_seq RESTART WITH 1;

-- Vérification finale
SELECT 'transaction' as table_name, COUNT(*) FROM transaction
UNION ALL
SELECT 'paiement', COUNT(*) FROM paiement
UNION ALL
SELECT 'dividende', COUNT(*) FROM dividende
UNION ALL
SELECT 'possession', COUNT(*) FROM possession
UNION ALL
SELECT 'revenus', COUNT(*) FROM revenus;
```

Toutes les tables doivent afficher `count = 0`.
